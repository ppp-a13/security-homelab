# Учебная Red Team / Blue Team лаборатория

---

Лаборатория для практики атак и защиты от них. Создал для отработки реальных сценариев по типу брутфорса, reverse shell и других. Также, чтобы учиться работать в SIEM и IDS.

## Стек

---

| Компонент | Инструмент | Статус       |
|---|---|--------------|
| Гипервизор | VMware Fusion Pro | Готово       |
| SIEM | Wazuh 4.14.5 | Готово       |
| IDS | Suricata | В процессе   |
| Атакующая машина | Kali Linux ARM64 | В процессе   |
| Жертва (Linux) | Ubuntu 26.04 + DVWA | В процессе   |
| Жертва (Windows) | Windows 11 ARM | В процессе   |
