# Архитектура лаборатории

---

## Хост

---

| Параметр   | Значение |
|------------|---|
| Устройство | MacBook Pro M3 Pro |
| RAM        | 18 GB |
| Гипервизор | VMware Fusion Pro |
| Архитектура | ARM64 (Apple Silicon) |

## Сеть

---

Машины находятся в изолированной host-only сети `vmnet2`.
У жертв нет доступа в интернет.
Wazuh SIEM имеет NAT-адаптер для выхода в интернет только для первоначальной установки пакетов.

---

| Машина | ОС | IP | CPU | RAM | Роль                               |

| kali | Kali Linux ARM64 | 10.10.10.10 | 2 | 3 GB | Атакующий                          |
| wazuh-siem | Ubuntu 26.04 ARM64 | 10.10.10.20 | 2 | 6 GB | Wazuh Manager + Indexer + Dashboard |
| dvwa-victim | Ubuntu 26.04 ARM64 | 10.10.10.30 | 2 | 3 GB | Пока не установил                  |

## RAM

---

Все машины одновременно никогда не запускаются из-за нехватки памяти.
Для себя выделил 2 рабочие комбинации:

| Сценарий | Машины | RAM    |
|---|---|--------|
| Linux-атаки | wazuh-siem + dvwa-victim + kali | ~12 GB |
| Windows-атаки | wazuh-siem + windows-victim + kali | ~13 GB |

## Снапшоты

---

Каждая машина имеет снапшот `clean` — состояние сразу после установки и настройки агентов.
Перед каждым новым сценарием жертву откатываю к данному снапшоту.
